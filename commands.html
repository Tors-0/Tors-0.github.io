Interactive Commands

Command Description
Commands copyright © 2020 by Viking Developer
vik echo Says hello to you
vik poll (@role/@user) someText Sends a poll to the mentioned user/group in the current channel
vik says someText Sends a fancy formatted message with the text after the command (ie. someText)

Moderator Commands

Command Description
Commands copyright © 2020 by Viking Developer
vik ban @user reason Borrow Thor's hammer to ban someone from your server. The user is permanently removed and cannot rejoin unless you unban them.
vik unban userID Summon the Bifrost to bring someone back from the dark realm of banned users. The user is now able to join your server again.
vik kick @user Have Vidar use his magic shoe to kick someone out of your server. They can still join again if they have an invite.
vik burn @user A user has died! This command will tell you how the community responds to their death. (NOTE: This is only for jokes please dont be mad about what comes from it)
vik embed someTitle~someText~someFooter Sends a custom embed message with a title, body, and footer. You can specify these elements by replacing someText with "hello world"
