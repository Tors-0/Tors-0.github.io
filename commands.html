Interactive Commands

Command
Description
vik echo
Says hello to you
vik poll (@group/@user) someText
Sends a poll to the mentioned group or user in the current channel
vik says someText
Sends a fancy styled message containing whatever you put in place of "someText"

Moderator Commands

Command
Description
vik ban @user reason
Borrow Thor's hammer to permanently ban a user from your server. They cannot rejoin unless you unban them
vik unban userID
Summon the bifrost to bring back a user from ban land. They can now join with an invite
vik kick
Have Vidar use his magic shoe to kick a user out of your server. They can rejoin with an invite
vik burn @user
Holds a traditional Viking funeral for the mentioned user. This is only for jokes please don't take the responses from this seriously
vik embed title~message~footer
Sends a custom embed message with the specified values. Replace each value with what you want the message to say, but leave the tildas in place or the message won't work

Utility Commands

Command
Description
vik serverinfo
Sends a detailed embed with server information, including creation date and member count
vik servers
Sends an embed with the bot's server count and total member count
vik uptime
Sends an embed with bot uptime and ping time
vik userinfo @user
Sends an embed with detailed information about the mentioned user
vik announce someText
Sends an announcement in the current channel containing someText
vik suggest sampleSuggestion
Sends a message to the main support server containing sampleSuggestion
vik bug report sampleReport
Sends a message to the main support server containing sampleReport
